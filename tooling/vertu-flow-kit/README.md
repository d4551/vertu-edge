# vertu-flow-kit

Bun/TypeScript tooling for Vertu FlowV1.

## Commands

- `vertu-flow validate <flow.yaml>`
- `vertu-flow compile <flow.yaml> [output.json]`
- `vertu-flow validate-model-manifest <manifest.json>`
- `vertu-flow doctor`

`vertu-flow doctor` is the typed host-readiness entrypoint for Bun/Swift/Brew/Python, Java 21, Android SDK/`sdkmanager`/`adb`, required contract files, and provider credential integrity.
